Authentication and authorization in .net mvc

Authentication
-It is a process of verification that verifies “Who you are” (it confirms that you are a valid (or invalid) user).
-it is a process to validation someone data against some data resource.
 (validate user corresponds to data which alreary store in database)

Authorization
-It is a process of verification that verifies “What to do” (It confirms you are permissible to do (or not to do) that).
-based on the role what we can access
-it is a security system which checks whether user has access to a particular resource
